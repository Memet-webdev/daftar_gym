Tahan & Geser ↕️

胸 Chest (Otot Dada)

Fokus: Membentuk dada bidang dan kekuatan mendorong.

1. Push-Up (Saran: 3 Set x 10 Reps)
2. Flat Bench Press (Saran: 3 Set x 12 Reps)

🦅 Back (Otot Punggung)

Fokus: Memperbaiki postur tubuh agar tegap & berbentuk V-Shape.

1. Lat Pulldown (Saran: 3 Set x 12 Reps)
2. Seated Cable Row (Saran: 3 Set x 10 Reps)

💪 Arms (Otot Lengan)

Fokus: Memperbesar otot lengan depan (Bicep) & belakang (Tricep).

1. Dumbbell Bicep Curl (Saran: 3 Set x 12 Reps)
2. Tricep Pushdown (Saran: 3 Set x 12 Reps)

🦵 Legs & Shoulder (Kaki & Bahu)

Fokus: Membakar kalori paling banyak dan memperkuat pondasi bawah.

1. Goblet Squat (Saran: 3 Set x 10 Reps)
2. Dumbbell Shoulder Press (Saran: 3 Set x 10 Reps)
